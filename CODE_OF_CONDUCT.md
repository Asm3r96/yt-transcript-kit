# Code of Conduct

## Our Standard

This project aims to be welcoming, respectful, and useful to contributors of all backgrounds and experience levels.

Examples of behavior that help create a positive environment:

- Being respectful in language and tone
- Assuming good intent while discussing technical disagreements
- Giving constructive feedback tied to code, docs, tests, or behavior
- Accepting correction gracefully

Examples of unacceptable behavior:

- Harassment, threats, discrimination, or personal attacks
- Trolling, baiting, or deliberately disruptive behavior
- Publishing private information without permission
- Repeatedly derailing technical discussions

## Scope

This Code of Conduct applies to repository discussions, issues, pull requests, and other project communication channels.

## Enforcement

If you experience or witness unacceptable behavior, report it to [asm3r96@gmail.com](mailto:asm3r96@gmail.com).

Reports will be reviewed promptly and handled as confidentially as practical.

Project maintainers may remove comments, close discussions, reject contributions, or block participation when needed to keep the project safe and productive.
