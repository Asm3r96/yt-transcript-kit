# Security Policy

## Supported Versions

We only provide security updates for the latest version of `yt-transcript-kit`.

| Version | Supported          |
| ------- | ------------------ |
| Latest release | :white_check_mark: |
| Older releases | :x: |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please do NOT open a public issue. Instead, please report it via the following method:

1. **Email**: [asm3r96@gmail.com](mailto:asm3r96@gmail.com)
2. **Details**: Provide a detailed description of the vulnerability, including steps to reproduce.

We will acknowledge your report within 48 hours and provide a timeline for a fix.

## Our Responsibilities

- We will investigate and verify the vulnerability.
- We will work on a fix.
- We will provide credit to the reporter (if desired) in the changelog.

## Scope

This policy applies to the source code within this repository. It does not apply to external services (like YouTube) that the library interacts with.
